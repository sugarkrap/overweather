# Overweather

The original goal of this project was to find out how the algorithm. Even if some of this code remain, and the long term goal of this project is getting back to reverse-engineer the matchmaking of OW2 (Or OW 2026 or whatever it is called later down the line), right now, it's a match register in which you can input your match results through simple screenshots. [More info here](dlx/project.md).

## How to build

It's an Angular app, run:

```bash
pnpm ng
```

And it's built.
